Active API Endpoints




NOTE: As more endpoints are tested they will be duely updated

For Requests other than GET request, Kindly use Postman

Companies Endpoints
  1. GET [domain_url]/companies
    an endpoint to get all companies that has benefitted from an MDA
  2. GET [domain_url]/companies/funds
    a temporary endpoint for @expenseng To get all funds received by companies
  3. GET [domain_url]/companies/search/:q
    where q is the search string used to look into the companies table.

MDAs Endpoints
  1. GET [domain_url]/mdas
    an endpoint to get all mdas and their profile
  2. GET [domain_url]/mdas/:id
    an endpoint to get an mda profile
  3. GET [domain_url]/mda/heads
    an endpoint to get all mda heads and their handles

Expenses Endpoints
  1. GET [domain_url]/expenses
    an endpoint to get all expenses, with mdas and companies
  2. GET [domain_url]/expenses/:id
    an endpoint to get an expense data
  3. GET [domain_url]/expenses/:year/:month
    an endpoint to filter expenses by year and month
  4. GET [domain_url]/expenses/total-monthly-payments
    an endpoint to get total monthly payments

Reports Endpoints
  1. GET [domain_url]/reports/sort/:month/:year
    an endpoint to sort report given month and year as parameters

Projects Endpoints
  1. GET [domain_url]/projects
    an endpoint to all projects and names of associated companies and mdas

Subscribers Endpoints
  1. GET [domain_url]/subscribers
    an endpoint to get all subscribers
  2. POST [domain_url]/subscribers
    an endpoint to subscribe for mails
    body:{"email": "yourname@doman.com"}
  3. DELETE [domain_url]/subscribers/:id
    an endpoint for a user to unsubscribe from getting mails with their :id
